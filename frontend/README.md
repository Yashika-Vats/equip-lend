# Equip Lend Application

## Project Overview
Equip Lend is a web-based application for managing equipment borrowing. Users can search, borrow, and return equipment. Admins can add or update equipment records. This project has two phases:
- **Phase 1:** Manual version
- **Phase 2:** AI-assisted version

## Features
- User authentication (login/logout)
- Equipment dashboard (search/filter)
- Borrow and return equipment
- Admin add/update equipment (Phase 1 manual)
- AI suggestions for categories and borrowing workflow (Phase 2 AI-assisted)

## Tech Stack
- **Frontend:** React.js
- **Backend:** Spring Boot (Java)
- **Database:** H2 / MySQL
- **APIs:** REST
